Supply Sentinel Azure Cloud Plan

外部シグナルを AI が構造化し、自社在庫・BOM・調達ルートに照合して業務影響へ翻訳する、早期警戒型サプライチェーン支援システム。

最終構成

Azure Container Apps公開 read-only API。max 1 replica。
Container Apps Jobcron 0 */6 * * * で6時間ごとに巡回。
Cosmos DB Serverlesslatest / run / alert を保存。
Storage Static Website3画面ダッシュボードを配信。
ACR BasicGitHub runner で build した image を保存。
GitHub Actions OIDCclient secret なしで Azure deploy。

データフロー

Container Apps Job -> news / supplier notice / inventory / BOM / routes / orders -> Azure OpenAI boundary (gpt-5.4 / gpt-5.4-mini) -> RiskEvent -> impact assessment -> Cosmos DB -> Container Apps API -> Static Web dashboard

AI の見せ方

入力 業界ニュースとサプライヤ通知の生テキスト。画面では「AIが読んだ外部テキスト」として表示。
構造化 対象材料、リスク種別、深刻度、遅延日数、割当率、対象期間、根拠を JSON として表示。

Azure OpenAI · gpt-5.4Sub-agent · gpt-5.4-miniRUN_MODE=demo

quota 通過前は deterministic mock で安定デモ。quota 通過後は `RUN_MODE=cloud` に切り替える。

Human-in-the-loop

AI が行うこと人が承認すること
情報収集、AIリスク抽出、影響評価、初動案作成、レポート生成発注変更、サプライヤ切替、顧客正式通知、価格改定、生産計画変更
